# Filename Big Bang 文件命名大爆炸

“大爆炸”是一种特殊的交互方式，可以分块儿选取文本。基于这种更符合直觉的颗粒度，文件命名或许也应适当调整，以快速挑出作者名、编号（案号）、年份、或出版社等文件名的指定部分。

出处：文字大爆炸：新交互方式倒推新文件命名思路》，发布时间未定。
