# Charty Rings 统计环

Apple Watch 的统计圆环，可以自定义任何数据，比如饮水量、咖啡因摄入或者财务状况。需配合 [Charty](https://chartyios.app) 使用。

- [Shortcuts 动作下载](https://www.icloud.com/shortcuts/72c69f16e0cc4bc5be4c0cb11a644249)
